LocalLoom Knowledge Base Testing

This is a test web page used for automated E2E testing of the web page import connector.

Section One: Overview

The web page connector fetches HTML content from any URL, extracts readable text, and indexes it into the knowledge base for RAG-powered search.

Section Two: How It Works

Jsoup parses the HTML document, splits content by heading sections, and creates one content fragment per section. Each fragment is embedded into ChromaDB for semantic search.

Section Three: Test Data

This page contains unique phrases like "quantum entanglement scheduler" and "recursive breadcrumb navigator" that can be used to verify search results match this specific document.
